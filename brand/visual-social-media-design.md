# Visual Social Media Design: NextMaven AI

Purpose: lightweight AI knowledge for creating NextMaven social media visuals. This is the reusable design direction, while sample-specific notes live in the Sample folder.

## Overall Style

Social media designs should feel modern, educational, and AI-first. Use clean layouts, strong topic hierarchy, and practical digital cues such as UI mockups, cursor arrows, app panels, workflow cards, tool icons, and soft tech gradients.

The design should help people understand the learning point quickly while still feeling polished enough for a professional AI education brand.

## Format Guidance

Use 16:9 for covers, YouTube thumbnails, landing banners, and ad-style hero creatives.

Use 4:5 for Instagram and Facebook feed posts where the content needs more vertical reading space.

Use 9:16 only for story, reel, TikTok, or shorts covers.

Keep important text inside a safe zone around the edges. Social platforms often crop previews, so avoid placing key words, logos, or faces too close to the border.

## Composition

Use one clear content hierarchy:

- Main hook or topic title
- Supporting method, benefit, or step label
- Visual proof or UI/tool context
- Optional bottom tag, category label, or CTA

Do not make every element compete. The headline should be the first thing people read; the image or UI mockup should explain the topic after that.

## Text Placement

Keep title text separated from busy visuals. If using a screenshot, dashboard, person, or interface mockup, place the main text in a clean zone above, beside, or below the visual.

Short text works best:

- 1 strong headline
- 1 short supporting line
- 1-2 small tags or labels

Avoid long paragraphs on social graphics. If the topic needs details, use a carousel or caption instead.

## Visual Elements

Use:

- Pill badges for step numbers, categories, or course labels
- UI cards for tools, prompts, checklists, and workflows
- Cursor arrow accents for digital interaction
- Simple icon marks for AI, automation, email, calendar, document, chat, or analytics topics
- Clean app mockups when explaining workflow or tool usage

Avoid:

- Random decorative icons
- Heavy text shadows
- Overcrowded dashboards
- Too many type styles
- Text directly over complex image areas

## Mood

The tone should be helpful, smart, and practical. It should feel like a useful AI learning post, not a generic tech poster.

For learning tips, use clean spacing and calm contrast.

For tool announcements or AI trends, use more motion, glow, and product-interface energy.

For case studies or results, show a clear before/after, metric, workflow path, or transformation idea.

## Quick Quality Check

Before accepting a social media design, check:

- Is the main idea readable in one second?
- Is the text inside the safe zone?
- Does the visual support the topic instead of decorating it?
- Are there no more than 2-3 text levels?
- Does it feel connected to AI learning, tools, workflow, or practical application?
- Would it still work when shown small in a feed?
